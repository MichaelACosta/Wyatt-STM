Detailed Description

template<typename Data_t>
class WSTM::WChannelReader< Data_t >

The read end of a multi-cast channel.

These objects are used to read messages that were written to a WChannel. The read operations are protected by STM transactions.

Note that the reader will start reading messages from the next message that is written to the channel after the reader is connected to the channel. The reader will not see any message written to the channel before it was connected.

Parameters
Data_tThe type of data sent through the channel. This type must be copyable. Normally this should be a std::shared_ptr<const Val_t> where Val_t is the actual data type being sent.
