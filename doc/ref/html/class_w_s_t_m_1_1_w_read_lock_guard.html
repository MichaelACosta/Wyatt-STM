Detailed Description

template<typename ReadLockable_t>
class WSTM::WReadLockGuard< ReadLockable_t >

Read-lock manager for WAtomic and WInconsistent.

When the lock is required just create one of these objects and the lock will be released when this object goes out of scope.

Parameters
ReadLockable_tThe type that will be read locked. Must have the methods readLock and readUnlock.
