Detailed Description

A multi-cast channel system built on top of the STM system.

This system is comprised of four main classes. The first is WChannel that creates the logical channel and is used to write messages to it. WChannelReader is used to read message from the channel. WChannelReader objects are connected to the logical channel either through the orginal WChannel object or via a WReadOnlyChannel object that is connected to the WChannel object. WReadOnlyChannel exists so that the ability to connect readers to a channel can be exposed in the public interface of classes without also exposing the ability to send messages. The final class is WChannelWriter which allows one to send messages through the channel. WChannel can already do this, but it maintains a strong reference to the logical channel, as does WChannelReader. WChannelWriter maintains a weak reference to the logical channel, so if the original WChannel and all the readers go away the logical channel will go away and WChannelWriter's Write method becomes a no-op preventing messages that will never be read from piling up.
